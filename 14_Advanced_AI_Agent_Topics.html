📌 Agentic Workflows kya hote hain?

Jab hum kehte hain "Agentic Workflow", toh hum ek aisi system ki baat karte hain jisme AI agent apne aap plan banata hai, tools use karta hai, output check karta hai, aur zaroorat padne par apni strategy bhi badal leta hai — bina human ke har step approve kiye!

Ek normal software program hamesha same steps follow karta hai. Lekin ek agentic workflow mein AI agent goal dekhta hai, rastaa khud dhundhta hai, mistakes se seekhta hai, aur tab tak kaam karta rehta hai jab tak goal achieve nahi ho jaata!

Autonomous Planning

Agent apne aap goal ko steps mein todta hai aur execute karta hai

Autonomous Planning — Poori Depth Mein

What is it? (Kya hai?)

Autonomous Planning woh process hai jisme ek AI agent ko sirf ek high-level goal diya jaata hai — jaise "Is company ki Q4 sales report banao" — aur agent khud:

  • Goal ko sub-goals mein todta hai
  • Har sub-goal ke liye steps plan karta hai
  • Steps execute karta hai
  • Results check karta hai
  • Plan adjust karta hai agar kuch galat ho

Why do we need it? (Kyun chahiye?)

Sochiye agar aap ek very complex task karna chahein — jaise "Mere liye ek business plan banao, market research karo, competitors analyze karo, aur ek presentation ready karo." Yeh kaam manually hazaron steps ka hai. Autonomous planning se ek agent yeh sab apne aap kar sakta hai!

How does it work? (Kaise kaam karta hai?)

Autonomous Planning Architecture

Step-by-Step Explanation

  1. Goal Input: User agent ko ek high-level goal deta hai. Jaise: "Mere competitors ka analysis karo aur report banao."
  2. Goal Decomposition: LLM Planner goal ko sub-tasks mein todta hai. Jaise: (1) Competitors list karo, (2) Har competitor ka data nikalo, (3) Compare karo, (4) Report banao.
  3. Dependency Mapping: Planner decide karta hai kaun sa task pehle hoga. Task 3 tabhi ho sakta jab Task 1 aur 2 complete hon.
  4. Tool Selection: Har task ke liye sahi tool select hota hai — web search, code execution, file writing etc.
  5. Execution: Tasks execute hote hain — sequentially ya parallel.
  6. Result Check: Agent output check karta hai — kya goal achieve hua? Agar nahi, plan revise hota hai.
  7. Final Output: Refined output user ko diya jaata hai.

# Plan-and-Execute Agent Pattern
# Goal: Ek complex task ko autonomously execute karna

from langchain.agents import AgentExecutor
from langchain_openai import ChatOpenAI
from langchain.tools import tool
import json

# ---- STEP 1: Tools define karo ----
@tool
def web_search(query: str) -> str:
    """Web pe kuch bhi search karo"""
    # Real implementation mein tavily/serpapi use hoga
    return f"Search results for: {query}"

@tool
def write_file(content: str, filename: str) -> str:
    """File mein content likhna"""
    with open(filename, 'w') as f:
        f.write(content)
    return f"File '{filename}' successfully likha gaya!"

# ---- STEP 2: Planner LLM ----
class AutonomousPlanner:
    def __init__(self):
        self.llm = ChatOpenAI(model="gpt-4o", temperature=0)
        self.tools = [web_search, write_file]
    
    def create_plan(self, goal: str) -> list:
        """Goal ko sub-tasks mein todna"""
        prompt = f"""
        Goal: {goal}
        
        Is goal ko 3-5 concrete steps mein todo.
        Har step specific aur executable hona chahiye.
        JSON format mein do: [{{"step": 1, "task": "...", "tool": "..."}}]
        """
        response = self.llm.invoke(prompt)
        return json.loads(response.content)
    
    def execute_plan(self, goal: str):
        """Plan banao aur execute karo"""
        print(f"🎯 Goal: {goal}")
        
        # Step 1: Plan banana
        plan = self.create_plan(goal)
        print(f"\n📋 Plan ({len(plan)} steps):")
        for step in plan:
            print(f"  Step {step['step']}: {step['task']}")
        
        # Step 2: Execute karna
        results = []
        for step in plan:
            print(f"\n⚙️ Executing Step {step['step']}...")
            # Har step ke liye agent run karo
            result = self._execute_step(step)
            results.append(result)
            print(f"  ✅ Done: {result[:50]}...")
        
        return results

# Usage:

planner.execute_plan("Top 5 AI companies ka analysis karo aur report banao")

Recursive Agents — Depth Mein Samjho

Kya Hota Hai Recursive Agent?

Recursive agent woh agent hai jo ek bada task solve karne ke liye apne aap ko recursively call karta hai — ek problem ko chhote chhote problems mein todta jaata hai jab tak solution milta nahi.

🌳

Tree Structure

Recursive agent ek tree banata hai — root problem → branches (sub-problems) → leaves (solutions).

🔁

Self-Calling Pattern

Jab ek problem bahut complex ho, agent nayi agent instance create karta hai us sub-problem ke liye.

Recursive Agent Tree

Real-World Example

Maano aapne agent ko bola: "Meri company ke liye ek complete AI strategy banao."

  • Main Agent → Sub-Agent 1: "Current AI landscape research karo"
  • Main Agent → Sub-Agent 2: "Competitors ki AI strategy analyze karo"
  • Main Agent → Sub-Agent 3: "Technical requirements define karo"
  • Sub-Agent 1 → Sub-Sub-Agent: "Latest AI papers dhundho"
  • Sab results merge → Final strategy document
Python — Recursive Agent Pattern
import asyncio
from dataclasses import dataclass
from typing import List, Optional

@dataclass
class Task:
    id: str
    description: str
    depth: int = 0
    max_depth: int = 3  # Infinite recursion se bachao!

class RecursiveAgent:
    """
    Recursive Agent jo complex tasks ko
    sub-tasks mein todta hai recursively
    """
    
    def __init__(self, llm):
        self.llm = llm
        self.results_cache = {}
    
    async def solve(self, task: Task) -> str:
        print(f"{'  ' * task.depth}🔍 Solving: {task.description[:40]}...")
        
        # Base case: Simple enough to solve directly
        if task.depth >= task.max_depth or self._is_simple(task):
            return await self._direct_solve(task)
        
        # Recursive case: Task ko sub-tasks mein todo
        sub_tasks = await self._decompose(task)
        
        # Parallel mein sub-tasks solve karo
        sub_results = await asyncio.gather(
            *[self.solve(Task(
                id=f"{task.id}.{i}",
                description=st,
                depth=task.depth + 1,
                max_depth=task.max_depth
            )) for i, st in enumerate(sub_tasks)]
        )
        
        # Merge all results
        return await self._merge_results(task, sub_results)
    
    def _is_simple(self, task: Task) -> bool:
        """Task simple hai ya complex? LLM se poocho"""
        # Complexity check logic
        return len(task.description) < 100
    
    async def _decompose(self, task: Task) -> List[str]:
        """Task ko sub-tasks mein todna"""
        prompt = f"Break down into 2-3 sub-tasks: {task.description}"
        # LLM call here
        return ["Sub-task A", "Sub-task B", "Sub-task C"]

⚠️ Common Mistake

Infinite Recursion! Hamesha max_depth set karo warna agent recursion mein ghus jaayega aur tokens aur time waste hoga. Production mein 3-5 levels se zyada recursion avoid karo.

Python — Browser Automation Agent
from playwright.async_api import async_playwright
from langchain.tools import tool
import asyncio

class BrowserAgent:
    """
    AI-powered browser agent jo websites control karta hai
    Bilkul ek human jaise!
    """
    
    def __init__(self):
        self.browser = None
        self.page = None
    
    async def setup(self):
        """Browser start karo"""
        p = await async_playwright().start()
        self.browser = await p.chromium.launch(headless=False)
        self.page = await self.browser.new_page()
    
    async def navigate(self, url: str):
        """URL pe jaao"""
        await self.page.goto(url)
        print(f"🌐 Navigated to: {url}")
    
    async def get_page_content(self) -> str:
        """Page ka text content nikalo (LLM ke liye)"""
        return await self.page.inner_text('body')
    
    async def click_element(self, selector: str):
        """Kisi element par click karo"""
        await self.page.click(selector)
        print(f"👆 Clicked: {selector}")
    
    async def fill_form(self, selector: str, text: str):
        """Form field fill karo"""
        await self.page.fill(selector, text)
        print(f"✍️ Filled {selector}: {text}")
    
    async def take_screenshot(self, path: str = "screenshot.png"):
        """Screenshot lo — AI vision ke liye useful!
        LLM screenshot dekh ke next action decide karta hai"""
        await self.page.screenshot(path=path)
        print(f"📸 Screenshot saved: {path}")
        return path
    
    async def ai_action(self, instruction: str, llm):
        """LLM ko screenshot dikhao aur next action poocho"""
        # Screenshot lo
        screenshot_path = await self.take_screenshot()
        content = await self.get_page_content()
        
        # LLM se poocho: "Is page par kya karna hai?"
        prompt = f"""
        Task: {instruction}
        
        Current page content (first 2000 chars):
        {content[:2000]}
        
        Next action batao:
        1. navigate(url) - kisi URL pe jaao
        2. click(selector) - element click karo
        3. fill(selector, text) - form fill karo
        4. done - task complete
        
        Format: action_name|parameter
        """
        
        decision = llm.invoke(prompt).content
        return decision

Python — Self-Debugging Code Execution Agent
import subprocess
import sys
import tempfile
import os

class CodeExecutionAgent:
    """
    Agent jo code likhta hai, run karta hai,
    errors dekhta hai, aur khud fix karta hai!
    """
    
    def __init__(self, llm, max_retries=3):
        self.llm = llm
        self.max_retries = max_retries
    
    def generate_code(self, task: str, previous_error: str = None) -> str:
        """LLM se code generate karo"""
        prompt = f"Write Python code to: {task}"
        if previous_error:
            prompt += f"\n\nPrevious error tha:\n{previous_error}\n\nPlease fix it."
        
        response = self.llm.invoke(prompt).content
        # Code extract karo response se
        return self._extract_code(response)
    
    def execute_code(self, code: str) -> dict:
        """Code ko safely execute karo"""
        # Temp file mein code likho
        with tempfile.NamedTemporaryFile(
            mode='w', suffix='.py', delete=False
        ) as f:
            f.write(code)
            temp_path = f.name
        
        try:
            # Run with timeout (30 seconds)
            result = subprocess.run(
                [sys.executable, temp_path],
                capture_output=True,
                text=True,
                timeout=30  # Safety!
            )
            return {
                "success": result.returncode == 0,
                "stdout": result.stdout,
                "stderr": result.stderr,
                "code": code
            }
        except subprocess.TimeoutExpired:
            return {"success": False, "stderr": "Timeout! Code 30s mein complete nahi hua"}
        finally:
            os.unlink(temp_path)
    
    def solve(self, task: str) -> str:
        """Main loop: generate → execute → debug → repeat"""
        error = None
        
        for attempt in range(self.max_retries):
            print(f"\n🔄 Attempt {attempt + 1}/{self.max_retries}")
            
            # Code generate karo
            code = self.generate_code(task, error)
            print(f"💻 Generated code:\n{code[:200]}...")
            
            # Execute karo
            result = self.execute_code(code)
            
            if result["success"]:
                print(f"✅ Success!\nOutput: {result['stdout']}")
                return result["stdout"]
            else:
                error = result["stderr"]
                print(f"❌ Error: {error[:100]}")
                print("🔧 Debugging aur fix karna...")
        
        return f"Max retries exceeded. Last error: {error}"

# result = agent.solve("Fibonacci sequence ke first 10 numbers calculate karo")

Python — Computer Use Agent (PyAutoGUI + Claude Vision)
import pyautogui
import anthropic
import base64
import time
from io import BytesIO
from PIL import Image

# Safety settings — IMPORTANT!
pyautogui.FAILSAFE = True   # Mouse corner pe emergency stop
pyautogui.PAUSE = 0.5       # Actions ke beech pause

class ComputerUseAgent:
    def __init__(self):
        self.client = anthropic.Anthropic()
    
    def take_screenshot(self) -> str:
        """Screenshot lo aur base64 mein convert karo"""
        screenshot = pyautogui.screenshot()
        buffer = BytesIO()
        screenshot.save(buffer, format='PNG')
        return base64.b64encode(buffer.getvalue()).decode()
    
    def ask_claude(self, task: str, screenshot_b64: str) -> dict:
        """Claude ko screenshot dikhao, next action poocho"""
        response = self.client.messages.create(
            model="claude-3-5-sonnet-20241022",
            max_tokens=1024,
            messages=[{
                "role": "user",
                "content": [
                    {
                        "type": "image",
                        "source": {
                            "type": "base64",
                            "media_type": "image/png",
                            "data": screenshot_b64
                        }
                    },
                    {
                        "type": "text",
                        "text": f"""
Task: {task}

Screen dekh ke decide karo kya karna hai.
JSON mein jawab do:
{{"action": "click|type|scroll|done", 
  "x": 100, "y": 200,  (click ke liye)
  "text": "hello",     (type ke liye)
  "reason": "why?"}}
                        """
                    }
                ]
            }]
        )
        import json
        return json.loads(response.content[0].text)
    
    def execute_action(self, action: dict):
        """Action execute karo"""
        if action["action"] == "click":
            pyautogui.click(action["x"], action["y"])
            print(f"👆 Clicked ({action['x']}, {action['y']})")
        elif action["action"] == "type":
            pyautogui.typewrite(action["text"], interval=0.05)
            print(f"⌨️ Typed: {action['text']}")
        elif action["action"] == "scroll":
            pyautogui.scroll(action.get("amount", -3))
    
    def run(self, task: str, max_steps=20):
        print(f"🖥️ Computer Use Agent starting...\nTask: {task}")
        
        for step in range(max_steps):
            screenshot = self.take_screenshot()
            action = self.ask_claude(task, screenshot)
            print(f"Step {step+1}: {action.get('reason', '')}")
            
            if action["action"] == "done":
                print("✅ Task Complete!")
                break
            
            self.execute_action(action)
            time.sleep(1)  # Page load ka wait
🚨 Danger — Safety First!

Computer use agents extremely powerful aur dangerous ho sakte hain. Hamesha: (1) Sandboxed VM use karo, (2) Sensitive data se door rakho, (3) Human approval implement karo, (4) Rate limiting lagao, (5) Actions log karo!

Interview Questions — Tool Use

🎤 Top Interview Questions

Q1: Browser automation aur computer use mein kya fark hai?
A: Browser automation sirf web browsers control karta hai (Playwright, Selenium). Computer use poora OS control karta hai — koi bhi app, system dialogs, desktop sab kuch.

Q2: Code execution agent mein infinite loop ka risk kaise handle karein?
A: Timeout (subprocess timeout), resource limits (memory/CPU caps), max iterations, aur sandboxed execution se.

Q3: Computer use agent ko production mein safely deploy kaise karein?
A: Isolated VM/container, least-privilege principle, human-in-the-loop approval for critical actions, complete audit logging.

AI Alignment

AI ko human values aur intentions ke saath align karna — ek fundamental challenge

Alignment Problem kya hai?

Alignment problem tab hota hai jab AI apne given objective ko pursue karta hai lekin human intent se deviate kar jaata hai. Ek classic example hai "paper clip maximizer" — agar AI ko sirf "maximize paper clips" ka goal diya jaaye, toh woh sab resources paper clips banana mein laga dega!

Sochiye aapne AI agent ko bola "Sales maximize karo." Agar properly aligned nahi hai, toh agent customers ko spam kare, misleading ads dikhaye, ya unethical tactics use kare — technically "sales badh" gayi lekin actually nahin chahiye tha!

Hallucination Reduction

LLMs jab galat facts confidently bolte hain — isko detect aur reduce karna

Hallucination kya hai?

AI hallucination tab hoti hai jab LLM aise facts, quotes, ya information generate karta hai jo bilkul galat hain lekin usse confident lagta hai ki sahi hain.

❌

Hallucination Example

User: "Einstein ne 1923 mein kya likha?"
AI: "Einstein ne 1923 mein 'Theory of Everything' publish ki" ← Yeh bilkul galat hai — aisi koi book nahi!

✅

Correct Response

AI: "Mujhe 1923 mein Einstein ki specific publication ka confident knowledge nahi hai. Please verify karo authoritative sources se."

Hallucination ke Types

TypeDescriptionExample
Factual HallucinationGalat facts confidently bolna"Paris France ki capital hai" ✅ but "Berlin Germany ki capital 1940 mein tha" ❌ galat detail
Source HallucinationFake citations ya references dena"Journal of AI 2021 page 45" — yeh journal exist hi nahi karta!
Temporal HallucinationTime-sensitive info ko mix karna2020 ki news ko 2024 ki event bata dena
Numerical HallucinationGalat numbers, statistics"India ki population 200 crore hai" ❌
Reasoning HallucinationGalat logical conclusionsCorrect premises se wrong conclusion nikaalna

Hallucination Reduce karne ke Techniques

  1. RAG (Retrieval Augmented Generation): LLM ko answer generate karne se pehle relevant documents retrieve karo. LLM apna answer un documents pe base kare.
  2. Chain of Thought Prompting: LLM ko step-by-step reasoning karne ke liye force karo. Isse logical errors pakad mein aate hain.
  3. Self-Consistency: Same question multiple baar poocho different temperatures se. Majority answer correct hota hai usually.
  4. Fact Checking Agent: Ek separate agent output verify kare — claims ko sources se match kare.
  5. Uncertainty Quantification: LLM ko "confidence score" dene ki training do. Low confidence → "I'm not sure" response.
  6. Grounding: Response ko specific sources se explicitly link karo — citations mandatory karo.
Python — Hallucination Detection System
from langchain_openai import ChatOpenAI
from langchain.prompts import ChatPromptTemplate
from pydantic import BaseModel
from typing import List

class HallucinationCheck(BaseModel):
    claim: str
    is_hallucination: bool
    confidence: float  # 0.0 - 1.0
    reason: str
    suggested_correction: str

class HallucinationDetector:
    """
    AI Agent jo doosre AI ke outputs mein
    hallucinations dhundhta hai
    """
    
    def __init__(self):
        self.judge_llm = ChatOpenAI(
            model="gpt-4o",
            temperature=0  # Deterministic for fact-checking
        )
    
    def extract_claims(self, text: str) -> List[str]:
        """Text se factual claims nikalo"""
        prompt = f"""
        Yeh text mein se saare factual claims nikalo
        jo verify kiye ja saken (dates, numbers, names, events):
        
        Text: {text}
        
        Sirf claims list karo, ek per line.
        """
        response = self.judge_llm.invoke(prompt)
        return response.content.split('\n')
    
    def check_claim(self, claim: str, context: str = "") -> HallucinationCheck:
        """Individual claim ko check karo"""
        prompt = f"""
        Yeh claim verify karo: "{claim}"
        
        Context: {context}
        
        JSON mein jawab do:
        {{
          "is_hallucination": true/false,
          "confidence": 0.0-1.0,
          "reason": "explanation",
          "suggested_correction": "agar galat hai toh sahi kya hai"
        }}
        """
        response = self.judge_llm.invoke(prompt)
        import json
        result = json.loads(response.content)
        return HallucinationCheck(claim=claim, **result)
    
    def analyze(self, ai_output: str) -> dict:
        """Complete analysis run karo"""
        claims = self.extract_claims(ai_output)
        checks = []
        hallucination_count = 0
        
        for claim in claims:
            if claim.strip():
                check = self.check_claim(claim)
                checks.append(check)
                if check.is_hallucination:
                    hallucination_count += 1
                    print(f"⚠️ Hallucination found: {claim}")
                    print(f"   Fix: {check.suggested_correction}")
        
        total = len(checks)
        return {
            "total_claims": total,
            "hallucinations": hallucination_count,
            "hallucination_rate": hallucination_count / total if total > 0 else 0,
            "checks": checks
        }

from dataclasses import dataclass
from typing import List, Callable
import re

@dataclass
class GuardrailResult:
    passed: bool
    violated_rule: str = None
    message: str = ""

class GuardrailSystem:
    """
    Multi-layer guardrail system
    Input → Guardrails → LLM → Output Guardrails → User
    """
    
    def __init__(self):
        self.input_guards = []
        self.output_guards = []
    
    # ---- INPUT GUARDRAILS ----
    
    def check_pii(self, text: str) -> GuardrailResult:
        """PII (Personal Info) detect karo"""
        patterns = {
            "email": r'\b[A-Za-z0-9._%+-]+@[A-Za-z0-9.-]+\.[A-Z|a-z]{2,}\b',
            "phone": r'\b\d{10}\b|\+91\d{10}',
            "aadhar": r'\b\d{4}\s\d{4}\s\d{4}\b',
            "credit_card": r'\b\d{4}[-\s]?\d{4}[-\s]?\d{4}[-\s]?\d{4}\b'
        }
        for pii_type, pattern in patterns.items():
            if re.search(pattern, text):
                return GuardrailResult(
                    passed=False,
                    violated_rule=f"PII Detected: {pii_type}",
                    message="Personal information share mat karo!"
                )
        return GuardrailResult(passed=True)
    
    def check_harmful_content(self, text: str) -> GuardrailResult:
        """Harmful keywords detect karo"""
        harmful_patterns = [
            r'\b(hack|exploit|malware|virus|bomb)\b',
            r'\b(illegal|illicit|smuggl)\b'
        ]
        for pattern in harmful_patterns:
            if re.search(pattern, text.lower()):
                return GuardrailResult(
                    passed=False,
                    violated_rule="Harmful Content",
                    message="Is request ko process nahi kar sakta."
                )
        return GuardrailResult(passed=True)
    
    def check_topic_relevance(self, text: str, allowed_topics: List[str]) -> GuardrailResult:
        """Topic check — sirf allowed domains mein kaam karo"""
        # LLM-based topic classification (simplified)
        text_lower = text.lower()
        is_relevant = any(topic in text_lower for topic in allowed_topics)
        
        if not is_relevant:
            return GuardrailResult(
                passed=False,
                violated_rule="Off-Topic",
                message=f"Main sirf {', '.join(allowed_topics)} ke bare mein help kar sakta hoon."
            )
        return GuardrailResult(passed=True)
    
    def run_input_guards(self, user_input: str) -> GuardrailResult:
        """Sabhi input guardrails run karo"""
        checks = [
            self.check_pii(user_input),
            self.check_harmful_content(user_input),
        ]
        for check in checks:
            if not check.passed:
                return check  # First failure pe rokk do
        return GuardrailResult(passed=True)

import pandas as pd
import numpy as np

class BiasDetector:
    """
    AI model ke decisions mein bias dhundho
    Example: Loan approval AI mein gender bias
    """
    
    def demographic_parity(self, predictions: pd.DataFrame,
                              protected_col: str, outcome_col: str) -> dict:
        """
        Demographic Parity check:
        Kya har group ko equal approval rate milta hai?
        """
        groups = predictions.groupby(protected_col)[outcome_col].mean()
        
        max_rate = groups.max()
        min_rate = groups.min()
        disparity = max_rate - min_rate
        
        print("\n📊 Demographic Parity Analysis:")
        for group, rate in groups.items():
            bar = '█' * int(rate * 30)
            print(f"  {group:15}: {bar} {rate:.1%}")
        
        is_biased = disparity > 0.1  # 10% threshold
        
        return {
            "group_rates": groups.to_dict(),
            "disparity": disparity,
            "is_biased": is_biased,
            "message": f"⚠️ BIAS DETECTED! {disparity:.1%} disparity" if is_biased else "✅ Fair"
        }
    
    def generate_bias_report(self, df: pd.DataFrame) -> str:
        """Complete bias audit report generate karo"""
        report = """
=== AI FAIRNESS AUDIT REPORT ===
Date: 2025-01-15
Model: Loan Approval AI v2.3

DEMOGRAPHIC PARITY CHECK:
"""
        gender_analysis = self.demographic_parity(df, 'gender', 'approved')
        report += f"Gender Disparity: {gender_analysis['disparity']:.1%}\n"
        report += f"Status: {gender_analysis['message']}\n"
        
        return report

import openai
import sounddevice as sd
import numpy as np
import scipy.io.wavfile as wav
import tempfile

class VoiceAgent:
    """
    Complete Voice Agent:
    Listen → Transcribe → LLM → Speak
    """
    
    def __init__(self):
        self.client = openai.OpenAI()
        self.sample_rate = 16000  # Whisper prefers 16kHz
        self.conversation_history = []
    
    def listen(self, duration=5) -> np.ndarray:
        """Microphone se audio record karo"""
        print("🎤 Bol dijiye... (5 seconds)")
        audio = sd.rec(
            int(duration * self.sample_rate),
            samplerate=self.sample_rate,
            channels=1, dtype='float32'
        )
        sd.wait()
        print("✅ Recording done!")
        return audio
    
    def transcribe(self, audio: np.ndarray) -> str:
        """Whisper se audio → text"""
        with tempfile.NamedTemporaryFile(suffix=".wav", delete=False) as f:
            wav.write(f.name, self.sample_rate, 
                      (audio * 32767).astype(np.int16))
            with open(f.name, 'rb') as audio_file:
                transcript = self.client.audio.transcriptions.create(
                    model="whisper-1",
                    file=audio_file,
                    language="hi"  # Hindi support!
                )
        print(f"📝 Aapne kaha: {transcript.text}")
        return transcript.text
    
    def think(self, user_text: str) -> str:
        """LLM se response generate karo"""
        self.conversation_history.append({
            "role": "user", "content": user_text
        })
        response = self.client.chat.completions.create(
            model="gpt-4o",
            messages=[
                {"role": "system", "content": "Aap ek helpful Hindi voice assistant hain. Short aur clear answers do."},
                *self.conversation_history
            ]
        )
        reply = response.choices[0].message.content
        self.conversation_history.append({
            "role": "assistant", "content": reply
        })
        return reply
    
    def speak(self, text: str):
        """OpenAI TTS se text → speech"""
        print(f"🔊 Agent bol raha hai: {text}")
        response = self.client.audio.speech.create(
            model="tts-1",
            voice="alloy",
            input=text
        )
        # Audio play karo (platform specific)
        with open("response.mp3", "wb") as f:
            f.write(response.content)
        # Play: os.system("afplay response.mp3")  # Mac
        # Play: os.system("mpg123 response.mp3")  # Linux
    
    def run_conversation(self):
        """Full voice conversation loop"""
        print("🎙️ Voice Agent Ready! Ctrl+C se band karo.\n")
        while True:
            audio = self.listen()
            text = self.transcribe(audio)
            response = self.think(text)
            self.speak(response)

import anthropic

class VisionAgent:
    """
    AI Vision Agent jo images analyze karta hai
    aur intelligent responses deta hai
    """
    
    def __init__(self):
        self.client = anthropic.Anthropic()
    
    def load_image(self, image_path: str) -> str:
        """Image ko base64 mein convert karo"""
        with open(image_path, 'rb') as f:
            return base64.b64encode(f.read()).decode()
    
    def analyze_image(self, image_path: str, question: str) -> str:
        """Image ke bare mein kuch bhi poocho"""
        image_data = self.load_image(image_path)
        # Extension se media type determine karo
        ext = Path(image_path).suffix.lower()
        media_type = {'.jpg': 'image/jpeg', '.png': 'image/png',
                       '.gif': 'image/gif', '.webp': 'image/webp'}.get(ext, 'image/jpeg')
        
        response = self.client.messages.create(
            model="claude-3-5-sonnet-20241022",
            max_tokens=1024,
            messages=[{
                "role": "user",
                "content": [
                    {
                        "type": "image",
                        "source": {
                            "type": "base64",
                            "media_type": media_type,
                            "data": image_data
                        }
                    },
                    {"type": "text", "text": question}
                ]
            }]
        )
        return response.content[0].text
    
    def extract_invoice_data(self, invoice_image: str) -> dict:
        """Invoice image se structured data nikalo"""
        prompt = """
        Is invoice se yeh information JSON mein nikalo:
        {
          "vendor_name": "",
          "invoice_number": "",
          "date": "",
          "total_amount": 0,
          "line_items": [{"description": "", "amount": 0}],
          "tax": 0,
          "currency": ""
        }
        Sirf JSON return karo, koi explanation nahi.
        """
        result = self.analyze_image(invoice_image, prompt)
        import json
        return json.loads(result)
    
    def detect_quality_issues(self, product_image: str) -> dict:

        2. Color/shape sahi hai?
        3. Overall quality score (0-10)?
        4. Accept/Reject recommendation?
        JSON format mein do.
        """)
        import json
        return json.loads(analysis)

            {"type": "function", "function": {
                "name": "pick_up",
                "description": "Kisi object ko utha lo",
                "parameters": {"type": "object", "properties": {
                    "object_name": {"type": "string"}
                }}
            }},
            {"type": "function", "function": {
                "name": "place_down",
                "description": "Current location par object rakh do",
                "parameters": {"type": "object", "properties": {}}
            }}
        ]
    
    def perceive_environment(self) -> str:
        """Current state ko string mein convert karo LLM ke liye"""
        return f"""
        Robot Status:
        - Position: {self.state.position}
        - Battery: {self.state.battery}%
        - Holding: {self.state.carrying_object or 'Nothing'}
        - Nearby objects: {', '.join(self.state.nearby_objects)}
        """
    
    def execute_command(self, human_command: str):
        """Natural language command ko robot actions mein convert karo"""
        environment = self.perceive_environment()
        print(f"\n🤖 Command: {human_command}")
        print(f"📍 Environment:\n{environment}")
        
        # LLM decide karta hai kaunse actions lene hain
        response = self.client.chat.completions.create(
            model="gpt-4o",
            messages=[
                {"role": "system", 
                 "content": "Tum ek warehouse robot ho. Available tools se kaam karo."},
                {"role": "user", 
                 "content": f"{environment}\nTask: {human_command}"}
            ],
            tools=self.tools,
            tool_choice="auto"
        )
        
        # Tool calls execute karo
        for tool_call in response.choices[0].message.tool_calls or []:
            fn_name = tool_call.function.name
            import json
            args = json.loads(tool_call.function.arguments)
            print(f"⚙️ Executing: {fn_name}({args})")
            # Actual robot hardware API call yahaan hoga
            getattr(self, f"_robot_{fn_name}", lambda **x: None)(**args)

AI-Native App kya hai?

Traditional apps mein AI ek feature hota tha — jaise "AI chatbot button". AI-native apps mein AI hi core experience hai — har interaction, har workflow AI-powered hota hai.

AI-Native App Design Principles

  1. AI as Collaborator: AI ek tool nahi, ek team member ki tarah behave kare. User ke context aur intent ko samjhe.
  2. Progressive Disclosure: Simple tasks ke liye AI automatically handle kare. Complex tasks par human control de.
  3. Memory & Personalization: App user ke past interactions, preferences yaad rakhe aur accordingly adapt kare.
  4. Multi-step Workflows: Single click se complete complex workflows trigger ho sakein.
  5. Transparent AI: User ko pata ho AI ne kya kiya aur kyun — explainability critical hai.
  6. Human-in-the-Loop: Critical decisions mein AI suggest kare, human approve kare.
🎤 Senior-Level Interview Questions

Q1: AI-native app aur traditional app with AI feature mein fundamental difference kya hai?
A: Architecture difference — traditional apps mein AI ek additional layer hai. AI-native apps mein AI ke aas paas poora app design hota hai. Data model, UX, backend sab AI workflows ke liye optimized hote hain.

Q2: Multimodal agent build karte waqt context window management kaise karein?
A: Images tokens zyada consume karti hain. Strategies: (1) Low-res thumbnails pehle, high-res only when needed. (2) Extracted text/captions use karo jab possible. (3) Selective context — sirf relevant history rakho.

Q3: Production mein robotics AI system mein safety kaise ensure karein?
A: (1) Physical constraint limits hardcode karo (max speed, force). (2) Human override hamesha available rakho. (3) Confidence threshold — low confidence par stop aur human ask karo. (4) Simulation testing se pehle real deployment.

📖 Glossary — Part 14 Key Terms

Agentic Workflow
AI agent-driven task execution system jahan agent autonomously plan banata hai, tools use karta hai, aur goals achieve karta hai bina step-by-step human guidance ke.
Autonomous Planning
High-level goal ko executable sub-tasks mein automatically todna aur execute karna. LLM planner + task executor ka combination.
Recursive Agent
Woh agent jo complex problems ko recursively sub-problems mein toda karta hai, har sub-problem ke liye naya agent instance create karke.
Self-Improving Agent
Agent jo apne past performance analyze karke automatically apni strategies, prompts, ya tools improve karta hai bina retraining ke.
Browser Automation
AI agent ki capability to control web browsers — navigate, click, fill forms, scrape data. Playwright aur Selenium popular tools hain.
Computer Use Agent (CUA)
Agent jo poora computer OS control kar sakta hai — screenshots le, mouse move kare, keyboard type kare — any application ko use kar ke.
Alignment
AI ko human values, intentions, aur preferences ke saath align karna. RLHF, Constitutional AI iske popular approaches hain.
Hallucination
LLM ka confident tone mein galat ya fabricated information generate karna. RAG aur fact-checking se reduce kiya jaata hai.
Guardrails
Safety mechanisms jo AI agent ke inputs aur outputs filter/validate karte hain. Input + Output + Action guardrails — multi-layer protection.
Multimodal Agent
Agent jo multiple modalities (text, image, audio, video) process kar sakta hai ek unified system mein.
Voice Agent
ASR (speech-to-text) + LLM + TTS (text-to-speech) pipeline jo natural spoken language mein interact karta hai.
Vision Agent
Agent jo images aur visual information process karta hai — object detection, document understanding, chart analysis.
AI-Native App
Application jahan AI sirf ek feature nahi balki core architecture hai — har interaction AI-powered, design AI workflows ke around.
RLHF
Reinforcement Learning from Human Feedback — human raters AI outputs rate karte hain, phir AI un preferences se seekhta hai.
Robotics Agent
AI system jo physical world mein sensors se data lete hain, LLM se decisions lete hain, aur actuators se physical actions karte hain.
