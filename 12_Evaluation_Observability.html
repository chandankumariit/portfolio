1.1 Human Evaluation — Insaani Nazar Se Jaanch

Human Evaluation mein actual log (humans) AI ke outputs ko read karke rate karte hain. Yeh sabse reliable method hai lekin sabse mehenga aur slow bhi hai.

Human Evaluation Kyun Zaroori Hai?

Kai cheezein hain jo machine measure nahi kar sakti — jaise kya response natural lag raha hai, kya tone appropriate hai, kya context samjha gaya hai. Inhe judge karne ke liye human brain chahiye.

💡
Real-World Analogy

Jaise ek editor manually manuscript padh ke decide karta hai ki writing acchi hai ya nahi — waise hi human evaluators LLM outputs padhte hain.

Step-by-Step: Human Evaluation Kaise Karein?

Step 1: Evaluation Criteria Define Karein

Pehle decide karein ki kya check karna hai — accuracy, tone, safety, helpfulness. Har criterion ke liye clear definition likhen.

Step 2: Test Set Banayein

Representative questions/prompts ka set banayein jo real-world use cases cover kare. Minimum 100-500 samples.

Step 3: Evaluators Train Karein

Annotators ko guidelines samjhayein, examples dikhayein, calibration session karein.

Step 4: Blind Evaluation Karein

Evaluators ko model naam nahi batayein (blind). Bias avoid hoga.

Step 5: Inter-Annotator Agreement Measure Karein

Multiple evaluators ke scores compare karein. Cohen's Kappa use karein agreement measure karne ke liye.

Step 6: Results Analyze Karein

Average scores, disagreement patterns, edge cases identify karein aur model improve karein.

⚠️
Common Mistakes in Human Evaluation

1. Evaluator bias — agar evaluator ko pata hai ki kaun sa model hai, woh bias ho sakta hai. 2. Small test set — 10-20 examples se accurate evaluation nahi hoti. 3. No guidelines — bina clear criteria ke har evaluator alag alag rate karega.

PythonHuman Evaluation Score Aggregation
# Human Evaluation Scores Aggregate karna
import numpy as np
from sklearn.metrics import cohen_kappa_score

# 3 evaluators ne 5 responses ko 1-5 rate kiya
evaluator_1 = [4, 5, 2, 4, 3]
evaluator_2 = [3, 5, 2, 5, 3]
evaluator_3 = [4, 4, 3, 4, 4]

# Average Score per response
all_scores = np.array([evaluator_1, evaluator_2, evaluator_3])
avg_scores = np.mean(all_scores, axis=0)
print(f"Average scores: {avg_scores}")
# Output: Average scores: [3.67 4.67 2.33 4.33 3.33]

# Inter-Annotator Agreement (Cohen's Kappa)
# Binary: 1=Good (score>=4), 0=Bad (score<4)
binary_1 = [1 if s >= 4 else 0 for s in evaluator_1]
binary_2 = [1 if s >= 4 else 0 for s in evaluator_2]

kappa = cohen_kappa_score(binary_1, binary_2)
print(f"Inter-Annotator Agreement (Kappa): {kappa:.2f}")
# Kappa > 0.6 = Good agreement
# Kappa > 0.8 = Excellent agreement

# Overall model score
model_score = np.mean(avg_scores)
print(f"Overall Model Human Eval Score: {model_score:.2f}/5")

BLEU Score — Kya Hai Aur Kaise Kaam Karta Hai?

BLEU (Bilingual Evaluation Understudy) ek metric hai jo measure karta hai ki model output aur reference text mein kitna overlap hai. Machine translation mein sabse zyada use hota hai.

PythonBLEU, ROUGE Scores
from nltk.translate.bleu_score import sentence_bleu, corpus_bleu
from rouge_score import rouge_scorer
import nltk

# ── BLEU Score ──
reference = [["the", "cat", "sat", "on", "the", "mat"]]
hypothesis = ["the", "cat", "is", "on", "the", "mat"]

bleu_score = sentence_bleu(reference, hypothesis)
print(f"BLEU Score: {bleu_score:.4f}")
# BLEU 0-1 scale. 1.0 = perfect match
# Good BLEU for translation: 0.5+

# ── ROUGE Score ──
scorer = rouge_scorer.RougeScorer(['rouge1', 'rouge2', 'rougeL'])

reference_text = "The AI model generated a helpful and accurate response."
hypothesis_text = "The AI generated a helpful response."

scores = scorer.score(reference_text, hypothesis_text)
print("\nROUGE Scores:")
for metric, score in scores.items():
    print(f"  {metric}: P={score.precision:.3f}, R={score.recall:.3f}, F1={score.fmeasure:.3f}")

# Output:
# rouge1: P=0.833, R=0.625, F1=0.714
# rouge2: P=0.500, R=0.375, F1=0.429
# rougeL: P=0.833, R=0.625, F1=0.714
🌟
Remember This!

BLEU = Precision-based (kitna model output reference mein hai) | ROUGE = Recall-based (kitna reference output mein captured hai). LLM evaluation ke liye ROUGE-L zyada useful hai kyunki yeh longest common subsequence dekhhta hai.

Judge Prompt Engineering

Judge LLM ko ek carefully designed evaluation prompt dete hain. Yeh prompt specify karta hai ki kya rate karna hai, kaise rate karna hai, aur kya format mein answer dena hai.

PythonLLM-as-a-Judge Implementation
from openai import OpenAI
import json

client = OpenAI()

def llm_judge(question: str, answer: str, criteria: list) -> dict:
    """
    LLM ko judge banata hai jo answer evaluate kare.
    """
    
    criteria_text = "\n".join([f"- {c}" for c in criteria])
    
    judge_prompt = f"""You are an expert evaluator. Evaluate the following answer.

Question: {question}

Answer to evaluate: {answer}

Evaluation Criteria:
{criteria_text}

For each criterion, provide:
1. Score (1-5)
2. Brief reasoning

Respond ONLY in JSON format:
{{
  "scores": {{
    "criterion_name": {{"score": X, "reasoning": "..."}}
  }},
  "overall_score": X,
  "verdict": "pass/fail",
  "feedback": "overall feedback"
}}"""

    response = client.chat.completions.create(
        model="gpt-4o",
        messages=[{"role": "user", "content": judge_prompt}],
        temperature=0  # Deterministic output
    )
    
    result = json.loads(response.choices[0].message.content)
    return result

# Usage example
result = llm_judge(
    question="What is machine learning?",
    answer="ML is a subset of AI where machines learn from data.",
    criteria=[
        "Accuracy: Is the answer factually correct?",
        "Completeness: Does it cover main points?",
        "Clarity: Is it easy to understand?"
    ]
)

print(json.dumps(result, indent=2))

PythonCustom Domain Benchmark
import json
from dataclasses import dataclass
from typing import List, Optional
from openai import OpenAI

@dataclass
class BenchmarkItem:
    question: str
    correct_answer: str
    category: str
    difficulty: str  # easy/medium/hard

class CustomBenchmark:
    def __init__(self, name: str, items: List[BenchmarkItem]):
        self.name = name
        self.items = items
        self.client = OpenAI()
        self.results = []
    
    def run(self, model: str = "gpt-4o-mini"):
        """Benchmark run karo model pe"""
        print(f"Running benchmark: {self.name}")
        
        for i, item in enumerate(self.items):
            response = self.client.chat.completions.create(
                model=model,
                messages=[{"role": "user", "content": item.question}],
                temperature=0
            )
            
            model_answer = response.choices[0].message.content
            correct = item.correct_answer.lower() in model_answer.lower()
            
            self.results.append({
                "question": item.question,
                "expected": item.correct_answer,
                "got": model_answer[:100],
                "correct": correct,
                "category": item.category,
                "difficulty": item.difficulty
            })
            print(f"  [{i+1}/{len(self.items)}] {'✓' if correct else '✗'} {item.category}")
    
    def report(self):
        """Detailed report generate karo"""
        total = len(self.results)
        correct = sum(1 for r in self.results if r["correct"])
        
        print(f"\n{'='*40}")
        print(f"BENCHMARK REPORT: {self.name}")
        print(f"Overall Accuracy: {correct}/{total} = {correct/total*100:.1f}%")
        
        # Category-wise breakdown
        categories = set(r["category"] for r in self.results)
        print("\nCategory Breakdown:")
        for cat in categories:
            cat_results = [r for r in self.results if r["category"] == cat]
            cat_correct = sum(1 for r in cat_results if r["correct"])
            print(f"  {cat}: {cat_correct}/{len(cat_results)} ({cat_correct/len(cat_results)*100:.0f}%)")

# Example benchmark for Python knowledge
benchmark = CustomBenchmark(
    name="Python Fundamentals",
    items=[
        BenchmarkItem("What is a list comprehension in Python?", "list comprehension", "Python", "easy"),
        BenchmarkItem("What does GIL stand for?", "Global Interpreter Lock", "Python", "hard"),
    ]
)
benchmark.run()
benchmark.report()

🧠
Precision vs Recall — Easy Trick

Precision = "Jo maine positive bola, mujhe kitna confidence tha?" (Yahan confidence = TP/(TP+FP)). Recall = "Jinhe main pakadna chahta tha, unhe maine kitna pakada?" Jaise police operation mein — Precision = pakde hue mein kitne sachchi criminals, Recall = sare criminals mein se kitne pakde.

PythonHallucination Detection — Self-Consistency Method
from openai import OpenAI
import re
from collections import Counter

client = OpenAI()

def detect_hallucination_self_consistency(
    question: str, 
    n_samples: int = 5,
    temperature: float = 0.7
) -> dict:
    """
    Self-consistency se hallucination detect karo.
    Agar alag alag runs mein alag answers aaye = hallucination risk.
    """
    
    answers = []
    
    for _ in range(n_samples):
        response = client.chat.completions.create(
            model="gpt-4o-mini",
            messages=[{
                "role": "user",
                "content": question + " Answer in one sentence."
            }],
            temperature=temperature
        )
        answers.append(response.choices[0].message.content.strip())
    
    # Check consistency: same core facts bolega reliable model
    # Simple: extract numbers/names and check if consistent
    consistency_score = _compute_consistency(answers)
    
    return {
        "question": question,
        "answers": answers,
        "consistency_score": consistency_score,
        "hallucination_risk": "HIGH" if consistency_score < 0.6 else "LOW"
    }

def _compute_consistency(answers: list) -> float:
    """Extract key entities aur check karein similarity"""
    # Simplified: count common words across answers
    word_sets = [set(a.lower().split()) for a in answers]
    common_words = word_sets[0]
    for ws in word_sets[1:]:
        common_words &= ws
    
    # Remove stop words
    stopwords = {'the', 'a', 'is', 'in', 'of', 'and', 'to'}
    meaningful_common = common_words - stopwords
    
    avg_words = sum(len(ws) for ws in word_sets) / len(word_sets)
    return len(meaningful_common) / avg_words if avg_words > 0 else 0

# Test karo
result = detect_hallucination_self_consistency(
    "What year was Python programming language created?"
)
print(f"Risk: {result['hallucination_risk']}")
print(f"Consistency: {result['consistency_score']:.2f}")

PythonLatency Measurement & Profiling
import time
import statistics
from openai import OpenAI
from dataclasses import dataclass

client = OpenAI()

@dataclass
class LatencyResult:
    ttft: float        # Time to first token
    total_time: float  # Total response time
    tokens: int        # Tokens generated
    tps: float         # Tokens per second

def measure_latency(prompt: str) -> LatencyResult:
    """LLM call ki latency accurately measure karo"""
    
    start = time.perf_counter()
    first_token_time = None
    total_tokens = 0
    
    # Streaming mode mein TTFT measure karo
    with client.chat.completions.create(
        model="gpt-4o-mini",
        messages=[{"role": "user", "content": prompt}],
        stream=True
    ) as stream:
        for chunk in stream:
            if first_token_time is None:
                first_token_time = time.perf_counter()
            
            if chunk.choices[0].delta.content:
                total_tokens += 1
    
    end = time.perf_counter()
    
    ttft = first_token_time - start
    total_time = end - start
    tps = total_tokens / total_time if total_time > 0 else 0
    
    return LatencyResult(ttft, total_time, total_tokens, tps)

# Multiple runs se percentiles nikalo
def benchmark_latency(prompt: str, n: int = 10):
    results = [measure_latency(prompt) for _ in range(n)]
    
    total_times = [r.total_time for r in results]
    ttfts = [r.ttft for r in results]
    
    print(f"Latency Report ({n} runs):")

    print(f"  Total - P50: {statistics.median(total_times):.3f}s | P99: {sorted(total_times)[-1]:.3f}s")
    print(f"  Avg TPS: {statistics.mean([r.tps for r in results]):.1f}")

PRICING = {
    "gpt-4o": {"input": 2.50, "output": 10.00},
    "gpt-4o-mini": {"input": 0.15, "output": 0.60},

    "claude-3-haiku": {"input": 0.25, "output": 1.25},

@dataclass
class CostTracker:
    """Real-time cost tracking for AI applications"""
    
    total_cost_usd: float = 0.0
    total_input_tokens: int = 0
    total_output_tokens: int = 0
    call_count: int = 0
    model_breakdown: Dict = field(default_factory=dict)
    
    def track(self, model: str, input_tokens: int, output_tokens: int):
        if model not in PRICING:
            print(f"Warning: Unknown model {model}")
            return
        
        pricing = PRICING[model]
        cost = (input_tokens * pricing["input"] + 
                output_tokens * pricing["output"]) / 1_000_000
        
        self.total_cost_usd += cost
        self.total_input_tokens += input_tokens
        self.total_output_tokens += output_tokens
        self.call_count += 1
        
        if model not in self.model_breakdown:
            self.model_breakdown[model] = {"cost": 0, "calls": 0}
        self.model_breakdown[model]["cost"] += cost
        self.model_breakdown[model]["calls"] += 1
        
        return cost
    
    def report(self):
        print(f"\n💰 Cost Report")
        print(f"  Total Cost: ${self.total_cost_usd:.6f}")
        print(f"  Avg per call: ${self.total_cost_usd/self.call_count:.6f}")
        print(f"  Total tokens: {self.total_input_tokens + self.total_output_tokens:,}")
        print(f"\n  Per Model:")
        for model, data in self.model_breakdown.items():
            print(f"    {model}: ${data['cost']:.6f} ({data['calls']} calls)")

2.5 Tool Success Rate — Agent Ke Haath Ki Taaqat

AI Agents tools (functions, APIs, code executors) use karte hain. Tool Success Rate = kitne percent tool calls successfully complete hue (bina error ke aur correct result ke saath).

PythonTool Call Monitoring System
import time
import functools
from collections import defaultdict
from typing import Callable

class ToolMonitor:
    """Agent ke tool calls ko monitor karta hai"""
    
    def __init__(self):
        self.stats = defaultdict(lambda: {
            "calls": 0, "success": 0, "fail": 0,
            "total_time": 0.0, "errors": []
        })
    
    def track(self, tool_name: str):
        """Decorator: kisi bhi function ko track karo"""
        def decorator(func: Callable) -> Callable:
            @functools.wraps(func)
            def wrapper(*args, **kwargs):
                stat = self.stats[tool_name]
                stat["calls"] += 1
                start = time.perf_counter()
                
                try:
                    result = func(*args, **kwargs)
                    stat["success"] += 1
                    return result
                except Exception as e:
                    stat["fail"] += 1
                    stat["errors"].append(str(e))
                    raise
                finally:
                    stat["total_time"] += time.perf_counter() - start
            
            return wrapper
        return decorator
    
    def report(self):
        print("\n🔧 Tool Success Rate Report")
        print(f"{'Tool':<20} {'Calls':<8} {'Success%':<12} {'Avg Time'}")
        print("-"*55)
        
        for tool, stat in self.stats.items():
            success_rate = stat["success"] / stat["calls"] * 100
            avg_time = stat["total_time"] / stat["calls"]
            icon = "✅" if success_rate >= 95 else "⚠️" if success_rate >= 80 else "❌"
            print(f"{icon} {tool:<18} {stat['calls']:<8} {success_rate:<11.1f}% {avg_time:.3f}s")

import logging
import json
import time
import uuid
from datetime import datetime
from typing import Optional

class LLMLogger:
    """
    Production-grade structured logger for LLM applications.
    PII (Personal Identifiable Info) ko mask karta hai.
    """
    
    def __init__(self, app_name: str):
        self.logger = logging.getLogger(app_name)
        self.app_name = app_name
        
        # JSON formatter setup
        handler = logging.StreamHandler()
        handler.setFormatter(logging.Formatter('%(message)s'))
        self.logger.addHandler(handler)
        self.logger.setLevel(logging.INFO)
    
    def _log(self, level: str, event: str, **kwargs):
        log_entry = {
            "timestamp": datetime.utcnow().isoformat() + "Z",
            "level": level,
            "app": self.app_name,
            "event": event,
            **kwargs
        }
        getattr(self.logger, level.lower())(json.dumps(log_entry))
    
    def log_request(self, session_id: str, model: str, 
                     prompt_length: int, user_id: Optional[str] = None):
        self._log("INFO", "llm_request",
            session_id=session_id,
            model=model,
            prompt_length=prompt_length,
            user_id=hash(user_id) if user_id else None  # PII masking
        )
    
    def log_response(self, session_id: str, model: str,
                      input_tokens: int, output_tokens: int,
                      latency_ms: float, success: bool):
        cost = (input_tokens * 0.15 + output_tokens * 0.60) / 1_000_000
        
        self._log("INFO" if success else "ERROR", "llm_response",
            session_id=session_id,
            model=model,
            input_tokens=input_tokens,
            output_tokens=output_tokens,
            latency_ms=round(latency_ms, 2),
            cost_usd=round(cost, 8),
            success=success
        )
    
    def log_error(self, session_id: str, error_type: str, message: str):
        self._log("ERROR", "llm_error",
            session_id=session_id,
            error_type=error_type,
            message=message
        )

import time
from collections import deque
from typing import Callable

class LLMMonitor:
    """
    Real-time monitoring with alerting for LLM applications.
    Rolling window mein metrics track karta hai.
    """
    
    def __init__(self, window_seconds: int = 300):  # 5 minute window
        self.window = window_seconds
        self.latencies = deque()
        self.errors = deque()
        self.requests = deque()
        self.alerts = []
        
        # Alert thresholds
        self.thresholds = {
            "p95_latency_ms": 5000,
            "error_rate": 0.05,   # 5%
            "min_rps": 0.1         # Alert if too few requests (service down?)
        }
    
    def _prune(self):
        """Old entries ko window se remove karo"""
        cutoff = time.time() - self.window
        while self.latencies and self.latencies[0][0] < cutoff:
            self.latencies.popleft()
        while self.errors and self.errors[0] < cutoff:
            self.errors.popleft()
        while self.requests and self.requests[0] < cutoff:
            self.requests.popleft()
    
    def record(self, latency_ms: float, success: bool):
        now = time.time()
        self.requests.append(now)
        self.latencies.append((now, latency_ms))
        if not success:
            self.errors.append(now)
        
        self._prune()
        self._check_alerts()
    
    def _check_alerts(self):
        if len(self.latencies) < 5: return
        
        # P95 Latency check
        sorted_lat = sorted(x[1] for x in self.latencies)
        p95 = sorted_lat[int(len(sorted_lat) * 0.95)]
        
        if p95 > self.thresholds["p95_latency_ms"]:
            self._alert(f"⚠️ HIGH LATENCY: P95={p95:.0f}ms (threshold: {self.thresholds['p95_latency_ms']}ms)")
        
        # Error rate check
        error_rate = len(self.errors) / len(self.requests)
        if error_rate > self.thresholds["error_rate"]:
            self._alert(f"🚨 HIGH ERROR RATE: {error_rate*100:.1f}% (threshold: 5%)")
    
    def _alert(self, message: str):
        alert = {"time": time.strftime("%H:%M:%S"), "message": message}
        self.alerts.append(alert)
        print(f"[ALERT {alert['time']}] {message}")
        # Production mein: Slack/PagerDuty/email bhejna

3.5 Token Tracking — Har Token Ka Hisaab

Token tracking ek specialized form of telemetry hai jo specifically LLM token usage monitor karti hai. Yeh cost control, rate limit management, aur context window optimization ke liye critical hai.

PythonAdvanced Token Tracking System
import tiktoken
from dataclasses import dataclass, field
from typing import List, Dict
from datetime import date

@dataclass
class TokenBudget:
    """Daily token budget manager"""
    daily_limit: int = 1_000_000  # 1M tokens/day
    used_today: int = 0
    today: date = field(default_factory=date.today)
    history: List[Dict] = field(default_factory=list)
    
    def check_and_consume(self, tokens_needed: int) -> bool:
        """Kya enough budget bachi hai? Consume karo agar haan."""
        if date.today() != self.today:
            self._reset_daily()
        
        if self.used_today + tokens_needed > self.daily_limit:
            remaining = self.daily_limit - self.used_today
            print(f"⚠️ Token budget exceeded! Need {tokens_needed}, have {remaining}")
            return False
        
        self.used_today += tokens_needed
        return True
    
    def _reset_daily(self):
        self.history.append({"date": self.today.isoformat(), "used": self.used_today})
        self.used_today = 0
        self.today = date.today()
    
    @property
    def usage_percent(self):
        return self.used_today / self.daily_limit * 100

class TokenCounter:
    """Count tokens BEFORE sending to API (save cost!)
    tiktoken use karta hai — OpenAI ka official tokenizer"""
    
    def __init__(self, model: str = "gpt-4o"):
        self.enc = tiktoken.encoding_for_model(model)
    
    def count(self, text: str) -> int:
        return len(self.enc.encode(text))
    
    def count_messages(self, messages: List[Dict]) -> int:
        """Chat messages ka total token count"""
        total = 0
        for msg in messages:
            total += 4  # Per message overhead
            total += self.count(msg["content"])
        total += 2  # Reply prefix
        return total
    
    def truncate_to_limit(self, text: str, max_tokens: int) -> str:
        """Text ko token limit mein rakho"""
        tokens = self.enc.encode(text)
        if len(tokens) <= max_tokens:
            return text
        return self.enc.decode(tokens[:max_tokens])

# Usage
counter = TokenCounter("gpt-4o")
budget = TokenBudget(daily_limit=100_000)

messages = [
    {"role": "user", "content": "Explain neural networks in detail."}
]

tokens_needed = counter.count_messages(messages) + 500  # + expected output
print(f"Tokens needed: {tokens_needed}")

if budget.check_and_consume(tokens_needed):
    print(f"Budget used: {budget.usage_percent:.1f}%")
    # Proceed with API call
else:
    print("Request blocked — budget exceeded!")

import pytest
from typing import List

# Simplified RAG system
class SimpleRAG:
    def __init__(self, docs: List[str], llm_client):
        self.docs = docs
        self.llm = llm_client
    
    def retrieve(self, query: str, k: int = 2) -> List[str]:
        """Simple keyword-based retrieval (real use embeddings)"""
        scored = [(doc, sum(1 for w in query.split() if w.lower() in doc.lower()))
                  for doc in self.docs]
        return [doc for doc, score in sorted(scored, key=lambda x: -x[1])[:k]]
    
    def answer(self, query: str) -> dict:
        context = self.retrieve(query)
        prompt = f"Context: {' '.join(context)}\n\nQuestion: {query}\nAnswer:"
        
        response = self.llm.chat.completions.create(
            model="gpt-4o-mini",
            messages=[{"role": "user", "content": prompt}]
        )
        
        return {
            "answer": response.choices[0].message.content,
            "retrieved_docs": context,
            "context_used": len(context) > 0
        }

4.3 Stress Testing — System Ko Hadh Tak Push Karo

Stress testing mein hum system ko extreme conditions mein test karte hain — high load, large inputs, concurrent requests. Yeh identify karta hai ki system kab break karta hai aur bottlenecks kahan hain.

PythonLLM API Stress Testing with asyncio
import asyncio
import time
import statistics
from openai import AsyncOpenAI
from dataclasses import dataclass
from typing import List

client = AsyncOpenAI()

@dataclass
class StressResult:
    success: bool
    latency_ms: float
    error: str = ""

async def single_request(prompt: str) -> StressResult:
    start = time.perf_counter()
    try:
        response = await client.chat.completions.create(
            model="gpt-4o-mini",
            messages=[{"role": "user", "content": prompt}],
            max_tokens=100
        )
        latency = (time.perf_counter() - start) * 1000
        return StressResult(True, latency)
    except Exception as e:
        latency = (time.perf_counter() - start) * 1000
        return StressResult(False, latency, str(e))

async def stress_test(
    concurrent_users: int = 10,
    requests_per_user: int = 5
) -> dict:
    """Concurrent users simulate karke stress test karo"""
    
    print(f"Stress Test: {concurrent_users} users × {requests_per_user} requests")
    total = concurrent_users * requests_per_user
    
    tasks = [
        single_request("What is 2+2?")
        for _ in range(total)
    ]
    
    start = time.time()
    results: List[StressResult] = await asyncio.gather(*tasks, return_exceptions=True)
    total_time = time.time() - start
    
    # Analyze results
    successes = [r for r in results if isinstance(r, StressResult) and r.success]
    failures = [r for r in results if isinstance(r, StressResult) and not r.success]
    
    latencies = [r.latency_ms for r in successes]
    
    report = {
        "total_requests": total,
        "success_count": len(successes),
        "failure_count": len(failures),
        "success_rate": len(successes) / total * 100,
        "total_time_s": round(total_time, 2),
        "rps": round(total / total_time, 2),
        "p50_latency_ms": statistics.median(latencies) if latencies else 0,
        "p95_latency_ms": sorted(latencies)[int(len(latencies) * 0.95)] if latencies else 0,
    }
    
    print(f"Results: {report['success_rate']:.1f}% success | {report['rps']} req/s | P95: {report['p95_latency_ms']:.0f}ms")
    return report
